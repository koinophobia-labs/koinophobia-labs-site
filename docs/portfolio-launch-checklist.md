# Portfolio Launch Checklist

1. Local test
   - Run `npm install`
   - Run `npm run build`
   - Run `npm run dev`
   - Verify `/`, `/#concepts`, `/demos/tattoo-studio`, `/demos/fitness-coach`, and `/demos/coffee-shop`

2. Screenshot capture
   - Run `npm run screenshots`
   - Confirm files exist in `outreach-screenshots/`
   - Use mobile demo screenshots for outreach attachments

3. Vercel deploy
   - Commit the targeted concept-site changes
   - Push to the branch connected to the existing Vercel project
   - Confirm Vercel builds successfully

4. Verify live URLs
   - https://koinophobia-labs.vercel.app/#concepts
   - https://koinophobia-labs.vercel.app/demos/tattoo-studio
   - https://koinophobia-labs.vercel.app/demos/fitness-coach
   - https://koinophobia-labs.vercel.app/demos/coffee-shop

5. Send first 10 emails
   - 3 tattoo shops
   - 3 fitness coaches or gyms
   - 3 coffee shops or restaurants
   - 1 warm/local contact
   - Track replies, objections, and clicks manually
